# bcftools

A wrapper around [bcftools v1.17](http://www.htslib.org/download/) with a few utility scripts for use with TinJasmine
germline caller.

## Tools

### `bcftools_normalize`

```
bcftools norm [-f REF] --multiallelics - --check-ref e -o output.norm.vcf.gz input.vcf.gz
```
